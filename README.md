# ⚡ QUICK INSTALLATION

## 🔹 Clone Repository

```bash
git clone https://github.com/sahillume/PrimeBot.git
```

## 🔹 Open Folder

```bash
cd PrimeBot
```

## 🔹 Install Packages

```bash
npm install
```

## 🔹 Start Bot

```bash
node index.js
```

---

# ⚙️ CONFIGURATION

Edit your `config.js`

```js
module.exports = {
  botName: "PrimeBot v1.0.0",
```
